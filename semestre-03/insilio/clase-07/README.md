## rieles y voltaje

si tenemos la alimentación de un circuito a partir de una batería de 9v, esto es, hay dos terminales, uno artificialmente es llamado tierra y se usa de referencia para el otro terminal, que es el que tiene voltaje de 9V.

no existe el voltaje en sí mismo, es una diferencia entre dos nodos.

los terminales positivos y negativos de la alimentación son llamados rieles, y lo que ocurre en un circuito ocurre dentro de este margen de operación.

el insilio no ocurre en sí mismo, es contextual.
